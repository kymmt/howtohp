このサイトでは横幅539px以下の時に一箇所だけ使ってるんですけど、残念ながらマルチラインには対応してないので、使いどころは難しかもしれないけど、一応メモ。

.truncate {
    width: 250px; /* 超えちゃいけない幅 */
    white-space: nowrap; /* 改行させない */
    overflow: hidden; /* ellipsis対応してないブラウザ対応 */
    text-overflow: ellipsis;
}

text-overflow: ellipsis;に関してはIE6からつかえる
