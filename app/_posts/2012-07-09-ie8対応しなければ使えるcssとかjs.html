:last-child

これは、floatで横並びにした要素の最後だけ、マージン消すとか、リストの最後の奴だけボーダー消すとかそういう時に便利なんですが、IE8はこれに対応してません。残念。:first-childは使えます。

listをfloatで横並びにしてに、10pxずつ右に余白とって真ん中寄せの場合、こんな感じになるかと。

ul {
    margin:0 auto;
}

li {
    float:left;
    margin-right:10px;
}

li:last-child {
    margin-right:0;
}
